## Summary

Nvidia and Catches created RealFit, a new AI tool that shows how clothes fit using physics for more realistic images. Big investors like Antoine Arnault support this to help reduce online clothing returns. The technology is already used by brands like Amiri and aims to make online shopping easier and more accurate.

## Highlights

- Despite all the attempts, virtual try-on is still far from being the norm: In an eMarketer survey, only 1.4 percent of 1,000 shoppers aged 18 to 65 use it regularly.
